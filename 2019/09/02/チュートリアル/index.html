格子机场自签名证书安装教程

说明

此教程目的是为本机场用户提供一个持久稳定的服务，之前我们所采用的证书是从Let’s Encrypt签发的一个免费的SSL加密证书，机场是小机场，我们实在没有多余的钱来用于购买高级证书，最近也可能因为机场主的绑定ip过于频繁，导致新的节点证书不能正常签发，加之免费证书三个月需要手动重新续的特性，我们决定更换自签名证书来维持机场的稳定

更换自签名证书有什么好处

  • 更好的稳定性
  • 一劳永逸的特性
  • 给以后的添加新节点免去不必要的麻烦

    本篇教程中包含的平台

  • 博主拥有的
    • Windows 10
    • macOS
    • Linux
    • Android
    • IOS
    • OpenWrt

证书下载链接

Tutorial

  • Windows 10
    • 打开下载好的crt证书文件

    • 点击安装证书进入向导

    • 进入下一步选择第二项

    • 点选图中圈出的那一项

    • 在此界面请确认是否与该图一致

    • 完成

  • IOS

    • 用Safari浏览器打开上面的证书链接

    • 下载完毕之后进入设置可以看到有一个下载的描述档（没有的去通用-下面的描述文件与设备管理里面找）点击右上角的安装

    • 随后进入关于本机-最下面的信任设定里面给新安装的证书打钩

安卓各ROM和版本间可能会有些出入，这里拿我手里的一台原版ROM的Sony Xperia Z5给演示一遍

  • Android

    • 使用任意浏览器打开上面的证书链接

    • 打开证书会提示你输入设备的解锁密码

    • 之后会出现这个界面，随意填写一个名字就ok，下面的选项基本默认不用动

  • OpenWrt

    • 下载上文贴好的证书文件，连接路由器的SSH端口，传输文件的方式有多种，这里不再多述，不会还请自行百度


      放入路由器的/etc/ssl/certs/目录即可，但是请注意此方法并不适用于非OP固件以及Linux
  • macOS
    • 下载上文贴好的证书文件双击打开，下面的选项应选取为系统

    • 找到刚刚安装好的证书并双击

    • 单击信任选项卡，把第一个选框改成始终信任

如何直观的知道证书是否安装并信任成功？

  • 打开我们的 网站首页
    能正常访问即表示成功
